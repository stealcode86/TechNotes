1) To edit a file, type nano, followed by the filename you want to edit in Cloud Shell:

nano test.txt

2) To save the file with your new message, press CNTRL+X, press Y, and then press ENTER.

3) To confirm that your file was updated correctly, run the following command:

cat test.txt

4) To edit a file you can do the following 

(nano, vim, emacs) or use the Cloud Shell code editor:
